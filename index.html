Tiny Run Router

Generate smooth running loops on real city streets, edit the route on an interactive map, then snap it back to the road network.

The backend fetches a walkable street graph from OpenStreetMap using osmnx, builds a custom graph model and searches for “nice” loops with a Dijkstra-based algorithm. A vanilla JS + Leaflet frontend lets the user drag vertices, rubber-band the shape and send edits back to the backend, which re-snaps the path to real roads and recalculates distance and elevation.
