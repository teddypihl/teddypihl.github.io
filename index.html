Tiny Run Router

Generate smooth running loops on real city streets, edit the route directly on the map, then snap it back to the road network.

  • Python · FastAPI · OSMnx · Leaflet
  • Custom graph routing, editable map UI

The backend downloads a walkable street network from OpenStreetMap using osmnx, builds an internal graph, and searches for “nice” loops using Dijkstra-based routing. The frontend is a vanilla JS + Leaflet app with a clean UI that lets you drag and tweak the route. Edits are sent back to the backend, which re-snaps the path to real roads and recalculates distance.
