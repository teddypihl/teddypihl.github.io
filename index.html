Interactive Metrics

PID Lab

Simulink-style PID playground in the browser.

Block-diagram editor where you drag blocks (Step, Controller, Plant, Scope), connect them, and run a first-order simulation with P/PI/PID control. The scope overlay shows the step response directly in-canvas with performance metrics (overshoot, rise time, steady-state error).

  • Diagram UX: drag, connect, select — fast feedback.
  • Simulation: responsive plotting and metrics.
  • Deploy: GitHub Pages.

Aero · Visualization · Three.js

Real-time Instanced

Wind Tunnel

Real-time airflow visualization around a simplified car + wings.

A lightweight wind tunnel sandbox for intuition: thousands of instanced particles advect through a procedural flow field. Includes deflection around a smooth car body, underfloor acceleration + suction (diffuser / ride height), wing downwash effects, turbulence, and a wake with a velocity deficit + counter-rotating vortex pair. Tuned for interactive performance and clarity (not full CFD).

  • Flow field components: body deflection, underfloor, wing downwash, wake/vortices.
  • Rendering: instanced streak particles for performance.
  • Parameters: wing angles, ride height, diffuser efficiency.

Maps · Algorithms · FastAPI

Routing OSM

Tiny Run Router

Generate running loops on real streets, then edit and snap back to the road network.

Backend fetches a walkable graph from OpenStreetMap (OSMnx), builds a graph model and searches for “nice” loops with a Dijkstra-based approach. Frontend (vanilla JS + Leaflet) lets you drag vertices, rubber-band the route, then re-snap to roads and recompute distance/elevation.

  • Graph + routing: loop search and path snapping.
  • Map UX: interactive edit + backend re-snap.
  • Deployment: Render + GitHub Pages.

Python · FastAPI · OSMnx · Leaflet · GitHub Pages · Render
